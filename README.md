# Personal Portfolio Website

## Project Overview

This is a simple Personal Portfolio Website created using HTML5.

The website contains:
- About Section
- Skills Section
- Contact Section
- Navigation Menu
- Profile Image
- Contact Form

## HTML Concepts Used

- HTML5 Structure
- Semantic Tags
- Headings
- Paragraphs
- Lists
- Images
- Links
- Forms
- Input Validation

## Setup Instructions

1. Install VS Code.
2. Create project folder.
3. Add index.html file.
4. Create images folder.
5. Add profile picture inside images folder.
6. Open index.html in browser.
